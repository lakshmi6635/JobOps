# JobOps - Career Pipeline Tracker

JobOps is a professional web application built to help job seekers organize, track, and analyze their job applications in one centralized dashboard.

## 🌟 Key Features
* **Live Dashboard:** Real-time metrics for total apps, interviews, and offers.
* **Company Hub:** Grouped views of all applications sent to specific organizations.
* **Smart Status Tracking:** Easily update application stages from 'Applied' to 'Selected'.
* **Responsive Design:** Clean, modern UI built with Bootstrap 5.

## 🛠️ Technology Stack
* **Framework:** Flask (Python)
* **Database:** SQLite with SQLAlchemy ORM
* **Frontend:** Jinja2, Bootstrap 5, Bootstrap Icons
* **Deployment:** Docker & Docker-Compose
